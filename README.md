# PracticeSimpleWeb
Use by: php, js, css, jade, gulp.

# Cloning

``` git clone https://github.com/Practices/PracticeSimpleWeb.git ```

``` cd PracticeSimpleWeb ```

``` git fetch --all ```

``` git checkout develop ```

# Installation
``` npm install ```

``` bower install ```
if bower not installed, run ``` npm install -g bower ```
